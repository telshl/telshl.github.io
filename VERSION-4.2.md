# Version 4.2 / Phase 1：首頁與導覽微調

本版在不改動 Version 4.1 技術清理與 SEO 基礎設定的前提下，針對首頁第一屏與主導覽進行微調，目標是讓第一次造訪者在三秒內理解「這是誰、研究什麼、為何重要」。

## 主要調整

1. 首頁英文 tagline 改為：
   - Media, Democracy, Platforms & Public Knowledge

2. 首頁中文定位句改為更凝練的版本：
   - 研究媒體制度、新聞業危機、平台治理、事實查核與 AI 時代的公共傳播秩序。

3. 首頁主要 CTA 順序調整為：
   - 研究主題
   - 代表著作
   - 公共評論

4. 首頁三個快速入口順序調整為：
   - 研究主題
   - 代表著作
   - 公共評論

5. 主導覽順序微調：
   - 首頁
   - 關於
   - 研究
   - 代表著作
   - 學術出版
   - 公共評論
   - 媒體與演講
   - 聯絡

6. 網站 description 同步更新，使 SEO 摘要與首頁定位一致。

## 設計原則

教授個人網站的第一權威來源應是研究，其次是著作，最後才是公共寫作。因此本版將「研究主題」提前，讓首頁與導覽都更符合學術身份與公共知識入口的雙重定位。
